Stabilité d'un corps rigide en rotation (Satellite)

Enseignant : Professeur Gherouat

Cahier des charges : Évaluation de la stabilité via les théorèmes de Lyapunov et linéarisation.

created by DAHANE AHMED LAMINE

Objectif de l'exercice

L'objectif de cet exercice est d'analyser la stabilité d'un système dynamique non linéaire modélisant la rotation d'un corps rigide (un satellite) dans l'espace, régit par les équations d'Euler.

Les équations d'Euler sont définies par le système d'équations différentielles suivant :

J1(dω1/dt) = (J2 - J3)ω2ω3 + u1
J2(dω2/dt) = (J3 - J1)ω3ω1 + u2
J3(dω3/dt) = (J1 - J2)ω1ω2 + u3

Où :

  • ω1, ω2, ω3 : Composantes du vecteur de vitesse angulaire.
  • u1, u2, u3 : Couples d'entrée (commandes).
  • J1, J2, J3 : Moments d'inertie principaux (constantes strictement positives).

Nous devrons utiliser la méthode de linéarisation (Jacobien) et l'approche directe de Lyapunov pour évaluer la stabilité autour de l'origine.

1. Linéarisation du système et discussion sur la stabilité

On pose u1 = u2 = u3 = 0. Le système autonome s'écrit :

dω1/dt = f1(ω) = [(J2 - J3) / J1] * ω2ω3
dω2/dt = f2(ω) = [(J3 - J1) / J2] * ω3ω1
dω3/dt = f3(ω) = [(J1 - J2) / J3] * ω1ω2

Calcul de la matrice Jacobienne (A)

Pour linéariser le système autour de l'origine d'équilibre xe = (ω1, ω2, ω3) = (0,0,0), nous calculons la matrice Jacobienne :

A = [∂fi / ∂ωj] évaluée à ω = 0

Calculons les dérivées partielles pour la première ligne (f1) :

  • ∂f1 / ∂ω1 = 0
  • ∂f1 / ∂ω2 = [(J2 - J3) / J1] * ω3
  • ∂f1 / ∂ω3 = [(J2 - J3) / J1] * ω2

Au point d'équilibre (0,0,0), toutes les composantes ωi sont nulles. Ainsi, chaque dérivée partielle devient nulle. La matrice Jacobienne est une matrice nulle :

A =
[ 0 , 0 , 0 ]
[ 0 , 0 , 0 ]
[ 0 , 0 , 0 ]

Discussion de la stabilité

L'équation caractéristique de la matrice A est det(λI - A) = λ3 = 0.
Les valeurs propres sont donc λ1 = λ2 = λ3 = 0.

Conclusion critique :

Puisque les valeurs propres ont des parties réelles strictement égales à zéro (elles sont sur l'axe imaginaire), nous sommes dans un cas critique (cas douteux de la première méthode de Lyapunov). La linéarisation ne nous permet pas de conclure sur la stabilité du système non linéaire. Il est obligatoire de passer par la deuxième méthode de Lyapunov (fonction d'énergie).

2. Stabilité via une fonction de Lyapunov appropriée

Pour démontrer la stabilité de l'origine pour u = 0, nous allons utiliser l'énergie cinétique de rotation du satellite comme fonction candidate de Lyapunov.

Choix de la fonction de Lyapunov V(ω)

V(ω) = (1/2) * [ J1ω12 + J2ω22 + J3ω32 ]

Vérifions les conditions :

  1. V(0) = 0 : Évident si ω = (0,0,0).
  2. V(ω) > 0 pour ω ≠ 0 : Vrai car les moments d'inertie Ji sont des constantes strictement positives, constituant une forme quadratique définie positive.

Calcul de la dérivée dV/dt le long des trajectoires

En dérivant V par rapport au temps, on obtient :

dV/dt = J1ω1(dω1/dt) + J2ω2(dω2/dt) + J3ω3(dω3/dt)

En remplaçant les dωi/dt par les équations du système :

dV/dt = ω1 [ (J2 - J3)ω2ω3 ]
        + ω2 [ (J3 - J1)ω3ω1 ]
        + ω3 [ (J1 - J2)ω1ω2 ]

En factorisant par ω1ω2ω3 :

dV/dt = ω1ω2ω3 [ (J2 - J3) + (J3 - J1) + (J1 - J2) ]
dV/dt = ω1ω2ω3 [ 0 ] = 0

Conclusion sur la stabilité

Puisque dV/dt = 0 ≤ 0 pour tout ω, la fonction V(ω) est non croissante. Selon le théorème de Lyapunov, l'origine est Stable au sens de Lyapunov.

La stabilité est-elle asymptotique ?

Non. Pour qu'elle soit asymptotique, il faudrait que dV/dt < 0 pour ω ≠ 0 (définie négative). Ici, la dérivée est strictement nulle. Physiquement, cela signifie qu'en l'absence de couples d'entrée ou de frottements, l'énergie cinétique du satellite se conserve. Les trajectoires décrivent des orbites fermées (ellipsoïdes d'inertie) autour de l'origine sans jamais y converger.

3. Stabilité Asymptotique Globale avec commande par retour d'état

Supposons maintenant que les couples d'entrée sont appliqués de manière proportionnelle à la vitesse : ui = -kiωi (avec ki > 0, ce qui agit comme un frottement visqueux ou une loi de commande stabilisante).

Nouveau système en boucle fermée

J1(dω1/dt) = (J2 - J3)ω2ω3 - k1ω1
J2(dω2/dt) = (J3 - J1)ω3ω1 - k2ω2
J3(dω3/dt) = (J1 - J2)ω1ω2 - k3ω3

Analyse avec la même fonction de Lyapunov

Nous reprenons notre fonction candidate V(ω) = (1/2) * [ J1ω12 + J2ω22 + J3ω32 ] (toujours définie positive).

Calculons la nouvelle dérivée dV/dt :

dV/dt = ω1 [ (J2 - J3)ω2ω3 - k1ω1 ]
        + ω2 [ (J3 - J1)ω3ω1 - k2ω2 ]
        + ω3 [ (J1 - J2)ω1ω2 - k3ω3 ]

Nous savons d'après la question 2 que la partie liée aux termes croisés s'annule :

dV/dt = 0 - k1ω12 - k2ω22 - k3ω32

Conclusion sur la stabilité globale

Étant donné que les constantes de retour d'état ki sont strictement positives :

  • dV/dt < 0 pour tout ω ≠ 0 (La dérivée est définie négative).
  • La fonction V(ω) est radialement non bornée, c'est-à-dire que V(ω) → ∞ lorsque ||ω|| → ∞.

D'après le théorème de Lyapunov, le système en boucle fermée est Globalement Asymptotiquement Stable (GAS).

Physiquement, la commande injecte une dissipation d'énergie dans le système. L'énergie cinétique va strictement diminuer jusqu'à ce que la vitesse de rotation du satellite s'annule complètement (ω = 0), quelle que soit sa condition initiale.

Conclusion Générale

L'étude des équations d'Euler du mouvement d'un satellite illustre parfaitement les limitations et la puissance des différents outils d'analyse des systèmes non linéaires :

  1. La linéarisation (Théorème de Poincaré-Lyapunov) : S'est avérée insuffisante pour conclure sur le système libre, nous plaçant dans un cas critique avec des pôles sur l'axe imaginaire.
  2. La méthode directe de Lyapunov : A permis de prouver de manière élégante que le système autonome est intrinsèquement stable (grâce à la conservation de l'énergie), sans pour autant converger vers le repos.
  3. La commande par retour d'état : L'ajout d'une commande proportionnelle modifie la dynamique de l'énergie, rendant le système globalement asymptotiquement stable. C'est le principe fondamental de la stabilisation des engins spatiaux.
